o que é o linux?




O sistema operacional serve para que todos os softwares funcionem direitinho, de acordo com o que se espera do hardware, além de não permitir que um programa interfira no outro O Linux é utilizado em diversas finalidades, como o desenvolvimento de software O Linux é um sistema operacional de código aberto.

Sobre




Foi lançado em 17 de Setembro de 1991, criado por Linus Torvalds Utiliza a linguagem de programação C, POSIX e Linguagem Assembly Tem como alvo: PC’s, Dispositivos Móveis, servidores, mainframes e super computadores Derivado do GNU e com o núcleo de Kernel Linux
